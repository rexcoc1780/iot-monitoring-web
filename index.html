Project documentation & system information

Latar Belakang Proyek

Website ini dibuat sebagai bagian dari Uji Kompetensi Keahlian (UKK) untuk mengimplementasikan kemampuan di bidang IoT dan pengembangan web secara terintegrasi. Sistem ini berfungsi sebagai antarmuka monitoring dan kontrol perangkat IoT berbasis ESP32 yang memantau kondisi lingkungan serta mengendalikan perangkat secara real-time. Data suhu, kelembapan, dan intensitas cahaya dibaca dari sensor, lalu dikirim melalui protokol MQTT ke server untuk disimpan dalam database dan ditampilkan pada dashboard web.

Tujuan utama proyek ini adalah menyediakan sistem monitoring dan kontrol yang ringan, real-time, serta dapat diakses melalui browser maupun dijadikan aplikasi mobile berbasis web, sehingga fleksibel digunakan di berbagai perangkat.

Fitur Sistem

  • Monitoring suhu real-time (DHT11)
  • Monitoring kelembapan udara (%RH)
  • Deteksi cahaya LDR (gelap/terang)
  • Kontrol 4 relay (ON/OFF secara individual dan global)
  • Peringatan suhu ≥ 31°C otomatis
  • Grafik historis real-time
  • Log debug MQTT / serial real-time
  • Riwayat lengkap + kolom relay R1–R4
  • Export CSV + filter data
  • Tema gelap 🌙 & terang ☀️
  • Responsif + hamburger menu mobile
